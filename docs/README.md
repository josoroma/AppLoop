# AppLoop Docs Images

This folder contains the product spec, live product screenshots, and historical wireframe PNGs used as visual references for AppLoop.

## Spec Documents

- [SPECS.md](SPECS.md) is the product planning spec with epics, user stories, acceptance scenarios, and task lists.

## How SPECS.md References These Images

[SPECS.md](SPECS.md) embeds the current product screenshots in its **Current Screens** section:

- [screenshot-projects.png](screenshot-projects.png): current projects dashboard screen.

![Current projects dashboard screen](screenshot-projects.png)

- [screenshot-builder.png](screenshot-builder.png): current project builder screen.

![Current project builder screen](screenshot-builder.png)

The spec also keeps the wireframe set as reference material by naming the range [wireframe-01.png](wireframe-01.png) through [wireframe-10.png](wireframe-10.png). These wireframes describe intended layout and behavior patterns; the screenshots show the current implemented UI.
